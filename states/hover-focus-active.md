# Hover, focus, active

{% embed url="https://zellwk.com/blog/style-hover-focus-active-states/?ck\_subscriber\_id=420572458" %}

```css
button {
  background-color: #dedede;
}

button:hover {
  background-color: #aaa;
}

button:focus {
  outline: none;
  box-shadow: 0 0 0 3px lightskyblue;
}

button:active {
  background-color: #333;
  border-color: #333;
  color: #eee;
}
```
